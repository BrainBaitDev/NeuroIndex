# NeuroIndex APT Repository

Official APT repository for NeuroIndex - High-performance in-memory database.

## Installation

```bash
echo "deb [trusted=yes] https://your-public-account.github.io/neuroindex-apt/debian stable main" | sudo tee /etc/apt/sources.list.d/neuroindex.list
sudo apt update
sudo apt install neuroindex
```

Visit https://your-public-account.github.io/neuroindex-apt/ for full documentation.

---

This repository contains only the binary packages. Source code is available at: https://github.com/BrainBaitDev/neuroindex
