Installation

Add the repository and install NeuroIndex with these simple steps:

bash # Add the repository echo "deb [trusted=yes] https://BrainBaitDev.github.io/neuroindex/debian stable main" | sudo tee /etc/apt/sources.list.d/neuroindex.list # Update the package list sudo apt update # Install NeuroIndex sudo apt install neuroindex

Usage

Essential commands to manage the NeuroIndex service:

bash # Start the service sudo systemctl start neuroindex # Enable automatic startup at boot sudo systemctl enable neuroindex # Check the service status sudo systemctl status neuroindex

Configuration Files

bash Configuration file: /etc/neuroindex/config.toml Data directory: /var/lib/neuroindex/
